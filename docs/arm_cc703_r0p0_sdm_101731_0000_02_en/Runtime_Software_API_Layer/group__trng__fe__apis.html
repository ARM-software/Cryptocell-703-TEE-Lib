)
Returns
CC_OK on success.
A non-zero value from cc_trng_error.h on failure.
Parameters
[in]entropySizeBitsThe required entropy size in bits.Size must be bigger than CC_TRNG_MIN_ENTROPY_SIZE, and smaller than CC_TRNG_MAX_ENTROPY_SIZE.
[out]pOutEntropyPointer to the entropy buffer.
[in]outEntropySizeBytesThe entropy buffer size in bytes. The size must be big enough to hold the required entropySizeBits.
